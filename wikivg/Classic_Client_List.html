Clients[edit source]

This is a rather incomplete list of Minecraft Creative clients currently in development. This page only lists clients written from scratch.

Name Author(s) Language License Threaded Features
Charged Miners ChargedMinersClassic Minecraft Viewer, can connect to classic servers. D, Lua GPLv2 No Ingame server list, chat & block picker.
ClassiCube UnknownShadow200 C New BSD No Lightweight, speedhacks, CPE support
CrossCraft-Classic IridescentRose, MotoLegacy C++ MPL2 No Lightweight, Supports Consoles (PSP, PS Vita, etc.)
VoxelSrv* Patbox TypeScript MIT No Works in browser. Ingame server list (based on MineOnline's)
